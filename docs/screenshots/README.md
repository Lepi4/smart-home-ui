# Screenshots

Файлы в этой папке используются в README ALLHA-3D.

Рекомендуемые имена:

- `overview-map.png` — общий план;
- `room-view.png` — комната;
- `settings.png` — настройки;
- `rooms-panel.png` — список комнат;
- `devices-panel.png` — список устройств.

При обновлении скриншотов сохраняйте те же имена файлов, чтобы ссылки в README не ломались.
